gitcode 코드리뷰! 이렇게 이용해보세요.
좋은 소스코드를 위한 첫 걸음!
gitcode에서는 개발자를 위한 다양한 코드리뷰 서비스를 제공합니다.
How to use gitcode
Github 계정으로 간편하게 가입하세요.
회원가입에 복잡한 인증이나 가입 절차가 없습니다.
Github 계정을 연결해서 간편하게 가입하세요.
01
여러 개발자의 다양한 의견을 받아보세요.
퍼블릭 코드리뷰는 오픈된 공간에서 소스코드를 공유하고
의견을 나눌 수 있는 공간입니다. 지금 나의 소스를 공유하고 여러 개발자들의 다양한 의견과 노하우를 들어보세요.
02
Branch를 선택하고 코드리뷰를 요청할 범위를 선택하세요.
선택한 브랜치의 전체 소스코드 또는 변경된 소스코드에 대한 부분만!
코드리뷰가 필요한 범위에 대해서 선택할 수 있습니다.
03
소스코드 저자에 대한 정보를 알려주세요.
다른 개발자 또는 전문가가 리뷰를 진행하는데 도움이 됩니다.
소스코드 저자의 기술 이해도(숙련도)는?
Newbie
이제 막 시작한 초보라 많은 조언이 필요해요.
Beginner
경험은 있지만 아직 스터디가 필요해요.
Intermediate
익숙하지만 더 많은 노하우가 필요해요.
Expert
실력과 노하우를 겸비한 전문가입니다.
04
소스코드 정보와 코드리뷰의 목적을 알려주세요.
소스코드에 대한 설명과 코드리뷰의 목적
(학습, 검증 등)을 기술해 주세요.
저자님의 코드리뷰 목적에 맞춰 코드리뷰가 진행됩니다.
05
gitcode 코드리뷰 페이지를 확인하세요.
코드리뷰 요청이 완료되면 gitcode의 코드리뷰 메뉴에서 회원님의
코드리뷰 요청 내역을 확인할 수 있습니다.
06
내 소스에 대해 어떤 의견이 달렸을까 궁금하시죠?
gitcode의 코드리뷰 메뉴에서 리뷰 내용을 확인할 수 있어요.
여러 개발자들 또는 전문가가 남긴 리뷰 내용을 확인하고 대화를 나눠 보세요.
나의 소스에 대한 리뷰가 등록되면, gitcode 알림과 이메일로
새로운 소식이 발송됩니다.
Github 를 통해서 리뷰 의견을 남길 수 있습니다.
그리고 이 내용은 gitcode 사이트를 통해서 확인할 수 있습니다.
07
코드리뷰가 도움이 되었나요? 리뷰에 대한 평가를 진행해주세요.
의견을 남겨준 여러 개발자 중 베스트 리뷰어를 선정합니다.
리뷰에 대한 평가와 소감을 남겨주세요.
활발한 코드리뷰 문화 만들기에 함께하세요.
더 좋은 개발을 위한 gitcode의 서비스가 추가될 예정입니다.

쉽고 간편하게 학습을 시작해보세요.
